# 🎨 Clone — Exercício de Reprodução de Interface

## 🎯 Objetivo
A pasta `Clone` é um exercício prático para reproduzir a interface visual de uma página web existente de forma básica, sem funcionalidades avançadas.

## 📁 Conteúdo Típico
| Item | Descrição |
|---|---|
| `index.html` | Página-alvo com estrutura a estudar e reproduzir |
| `img/` | Pasta de imagens (caso exista) |

## 🔧 Como Usar
1. 🌐 Abra `index.html` no navegador para ver o ponto de partida
2. 🔍 Compare visualmente com a página original desejada
3. 📝 Copie a estrutura HTML (títulos, parágrafos, listas, imagens)
4. ⚙️ Evite scripts ou CSS complexos — mantenha simples!
5. 🖼️ Substitua textos e imagens para praticar

## 📚 Conceitos a Aprender
| Conceito | Detalhes |
|---|---|
| Estrutura básica | `<html>`, `<head>`, `<body>` |
| Elementos de texto | `<h1>`, `<p>`, `<ul>`, `<li>` |
| Imagens | Tag `<img>` e caminhos relativos |
| Layout simples | Sem CSS avançado — apenas HTML puro |
